## Prompt B

Create a 30-day learning roadmap.

Context:
- Current Situation: [Student/Professional/Freelancer]
- Current Skills: [Add Skills]
- Goal: [Target Goal]
- Available Time: [Hours per Day]
- Experience Level: [Beginner/Intermediate]
- Preferred Learning Style: [Videos/Projects/Reading]

Include:
- Weekly milestones
- Daily tasks
- Resources
- Projects
- Final outcome

Make it practical and beginner-friendly.

Compare both outputs and identify:
1. Which roadmap feels more personalized?
2. Which roadmap would you actually follow?
3. What role did context play in improving the result?

## Comparison

- Prompt A produced a generic response.
- Prompt B generated a more personalized answer.
- Providing context improved the quality of the response.

---

## Key Learnings

- Context greatly affects AI responses.
- Personalized prompts produce more relevant outputs.
- Starting a new conversation prevents previous context from influencing results.
