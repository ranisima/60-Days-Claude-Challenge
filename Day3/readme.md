# Day 3 - Role-Based Prompting

## Objective

Learn how role-based prompting changes AI responses.

---

## Prompt 1 (No Role)

How can I improve a mobile app for online shopping?

## Prompt 2 (Founder)

You are a startup founder.

How can I improve a mobile app for online shopping?

## Prompt 3 (Developer)

You are a senior software developer.

How can I improve a mobile app for online shopping?

## Comparison

- General response was broad.
- Founder response emphasized business strategy.
- Developer response emphasized technical implementation.

---

## Learnings

- Role-based prompting improves response quality.
- Different personas provide different perspectives.
- Choosing the right role helps generate more relevant answers.
